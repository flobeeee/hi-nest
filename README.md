- 목적 : Typescript + NestJS 공부
- 설명 : 노마드코더/ NestJS로 API 만들기
- 짧은 후기
  - 앞으로 API 서버 구축할 때 NestJS 필수로 써야겠다는 생각을 했다. (구조, 안정감, 좋은 옵션등 이점 多)
  - postman 대신에 insomnia 를 사용했다. UI가 깔끔해서 자주 써야겠다고 생각했다.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
